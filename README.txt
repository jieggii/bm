bm - simple bookmarking tool (literally for books).

Usage:
	bm [--help/-h] <command> [--help/-h] [args...]

Commands:
	new <title>        create a new book
	show <id>          show page number of the book <id>
	ls                 list all books
	set <id> <page>    set page number of book <id> to <page>
	rm <id>            remove book
